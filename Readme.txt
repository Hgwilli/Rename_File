To run 
1.) Install python, make sure to enable PATH https://www.python.org/downloads/
2.) Open Python IDLE (can be opened through start menu)
3.) File -> Open to open rename_files.py 
	Under # Example usage:
	4.) Choose your folder path by replacing (\\path\\to\\folder) with desired folder path
		Make sure to separate the parts of the path with \\ not \
	5.) Set the name by replacing (x) with desired name
	6.) Set the number you would like the counting to start at (1 by default)
7.) Save changes and click run (f5)
